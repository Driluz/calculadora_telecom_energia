Este projeto é uma coleção de aplicações web interativas desenvolvidas para auxiliar consultores de vendas a demonstrarem o valor, as vantagens e a economia de diferentes serviços para seus clientes de forma rápida e visual.

## 📂 Módulos do Projeto

Atualmente, o ecossistema conta com duas ferramentas principais. Clique nos links abaixo para acessar a documentação detalhada e o código de cada uma:

* [**💡 EcoSimulador (Energia)**](./simulador-energia/)
  Um simulador dinâmico que calcula a projeção de economia mensal e anual na conta de luz ao aderir a um plano de energia limpa com desconto fixo.

* [**📱 Comparador Mobile (Telecom)**](./comparador-internet/)
  Uma ferramenta inteligente que compara planos de internet tradicionais com planos modernos (com sistema de Rollover), calculando o acúmulo de dados e projetando a vantagem a longo prazo.

## 💻 Tecnologias e Boas Práticas

Todos os módulos foram construídos com foco em performance, responsividade (Mobile First) e Clean Code, utilizando tecnologias nativas:

* **HTML5:** Semântica e estruturação lógica.
* **CSS3:** Flexbox, Grid Layout, variáveis e animações fluidas.
* **JavaScript (Vanilla):** Manipulação de DOM, regras de negócio complexas (cálculos de datas e projeções temporais) e formatação de dados.
